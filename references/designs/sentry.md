# Sentry

暗色监控调试美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #0a0a0c | 温暖的深色监控屏 |
| Surface-1 | #181820 | 面板与仪表层 |
| Surface-2 | #262630 | 次级功能区 |
| Hairline | #343440 | 监控分割线 |
| Accent | #424050 | 克制的紫色强调 |
| Ink | #f7f5f0 | 温暖的米白文字 |
| Ink-muted | #a0a090 | 辅助的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 600，如监控面板标题般清晰。正文 DM Sans，字重 400，代码用等宽字体。整体如监控系统，暗色且专业。

## Visual Personality

- **监控暗色**——如服务器监控屏
- **调试专业**——错误追踪的精准感
- **紫色克制**——品牌色极少量点缀
- **开发者导向**——为程序员设计
- **技术权威**——监控工具的专业感

## Do

- 使用监控面板的界面布局
- 保持温暖的暗色背景
- 紫色仅用于错误状态，极少量
- 模仿监控工具的视觉层次

## Don't

- 过度使用紫色破坏专业感
- 使用非监控相关的彩色装饰
- 破坏监控面板的结构化布局
- 失去错误追踪的专业权威感
