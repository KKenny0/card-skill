# Spotify

暗色影院感的音乐美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #121212 | 沉浸的暗色舞台 |
| Surface-1 | #1e1e1e | 专辑与播放器层 |
| Surface-2 | #282828 | 次级导航区 |
| Hairline | #333333 | 细致的分割线 |
| Accent | #1db954 | 标志性的绿色点缀 |
| Ink | #ffffff | 纯净的白色文字 |
| Ink-muted | #a0a0a0 | 低调的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 700，如专辑封面般有力。正文 DM Sans，字重 400-500，清晰易读。整体如音乐播放器般流畅，暗色中透出律动。

## Visual Personality

- **影院沉浸**——暗色背景营造舞台感
- **音乐律动**——流动的视觉节奏
- **绿色克制**——标志性色仅1-2处点缀
- **暗色层次**——深浅变化的黑色空间
- **播放器美学**——如专业音频软件界面

## Do

- 使用暗色渐变营造深度
- 绿色仅用于核心交互，极少量
- 保持暗色调，白色为辅
- 模仿音乐播放器布局结构

## Don't

- 过度使用绿色品牌色
- 使用彩色装饰元素
- 破坏暗色的沉浸感
- 添加非音乐相关的视觉元素
