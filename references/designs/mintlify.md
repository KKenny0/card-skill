# Mintlify

极简文档手册美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #fafaf8 | 温暖的手册纸张 |
| Surface-1 | #f0f0e8 | 代码与示例层 |
| Surface-2 | #e6e6d8 | 导航与次级区 |
| Hairline | #dcdcc8 | 精准的分割线 |
| Accent | #1a1a18 | 极简的黑色强调 |
| Ink | #1c1c1a | 清晰的深灰文字 |
| Ink-muted | #8a8a78 | 柔和的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 600，如手册章节标题般清晰。正文 DM Sans，字重 400，如印刷文本般易读。代码用等宽字体，如技术手册般精准。

## Visual Personality

- **印刷手册**——如技术手册般严谨
- **双模友好**——深浅两色都清晰
- **文档结构**——章节与层级分明
- **极简克制**——去除所有干扰元素
- **易读至上**——排版为可读性服务

## Do

- 使用清晰的手册结构布局
- 保持温暖的白色背景
- 黑色用于关键术语或代码
- 强调文档的层级关系

## Don't

- 使用任何彩色装饰
- 破坏手册的结构化排版
- 过度使用黑色破坏简洁
- 添加非文档相关的视觉元素
