# Resend

极致纯净的黑白编辑风格

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #000000 | 深邃的午夜黑背景 |
| Surface-1 | #1a1a1a | 卡片与区块的微亮层 |
| Surface-2 | #2a2a2a | 次级元素的暗色层 |
| Hairline | #3a3a3a | 细致的分隔线 |
| Accent | #fcfdff | 极简的白色高光 |
| Ink | #fcfdff | 纯净的白色文字 |
| Ink-muted | #b0b0b0 | 低调的灰色文字 |

## Typography Vibe

标题使用 DM Serif Display，营造杂志感的优雅。正文用 DM Sans，字重 400-500，字间距微调至 -0.02em。整体如诗般流畅，留白极多。

## Visual Personality

- **极致留白**——空间即语言，负面积传达自信
- **单色纯粹**——拒绝任何装饰，只有黑与白
- **编辑质感**——如高端杂志般精致排版
- **技术克制**——每个元素都为功能服务
- **视觉静默**——安静而有力的存在感

## Do

- 使用大标题衬线字体搭配大留白
- 保持极简，减少装饰元素
- 白色仅用于关键高光，1-2处即可
- 保持文字层级清晰明确

## Don't

- 添加任何装饰性颜色
- 使用过多白色高光
- 破坏单色调的纯粹性
- 过度装饰或添加非必要元素
