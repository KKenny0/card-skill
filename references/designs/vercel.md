# Vercel

纯黑画布（#000），极致对比度。品牌三角标识只用白色。视觉语言：终端界面、开发者文档、技术宣言——黑底白字的工程美学。

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #000000 | 纯黑（Vercel 是少数允许纯黑的品牌） |
| Surface-1 | #111111 | 卡片、面板 |
| Surface-2 | #171717 | hover、次级面板 |
| Hairline | #333333 | 分隔线、边框 |
| Accent | #ffffff | 白色即强调——品牌三角用白 |
| Ink | #ededed | 主文字（近白） |
| Ink-muted | #888888 | 次要文字、标注 |
| Blue-accent | #0070f3 | 仅用于链接/CTA，非装饰 |

## Typography Vibe

Geist Mono / Geist Sans——Vercel 自家字体。等宽与无衬线混排，标题不花哨。排版感觉：终端输出、README 文档、工程笔记。

推荐 web 字体：Geist Sans（如不可用则 JetBrains Mono + Inter 组合）。标题 weight 600，正文 400。等宽用于代码和数据。

## Visual Personality

- **极致黑白对比**——黑底白字，不需要中间色调
- **三角标识**是唯一品牌元素——Vercel ▲ 用纯白
- **不使用渐变**——Vercel 界面几乎找不到渐变
- **等宽字体在正文区域自然出现**——代码片段、数据、路径名
- **圆角 6-8px**，克制
- **内容密度适中**——比 Stripe 密但比 Apple 密度更高

## Do

- 白色作为主要视觉锚点（标题、三角、关键数字）
- 蓝色 (#0070f3) 只用于链接/交互元素
- 代码/数据区域用等宽字体
- 用 surface 梯度制造层级（#000 → #111 → #171717）
- 信息分组用 hairline 分隔线

## Don't

- 不加阴影——纯黑底上阴影无意义
- 不用渐变背景
- 不使用彩色装饰——Vercel 的美来自克制
- 不把蓝色当装饰色——它是功能色
- 不用衬线体——严格无衬线 + 等宽
