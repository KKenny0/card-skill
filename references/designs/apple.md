# Apple

大面积留白，极简结构。信息密度极低——每个元素都有足够的呼吸空间。视觉语言：产品手册封面、画册、高端杂志内页。

| Role | Hex | Usage |

## Typography Vibe

SF Pro 风格——干净几何无衬线体。标题字重梯度大（700→400），标题与正文形成极端对比。排版感觉：空间即设计，文字是建筑。

推荐 web 字体：DM Sans 或 Plus Jakarta Sans（weight 700 标题, 400 正文）。标题可适度负字距 -0.02em。大标题允许极端字重对比。

## Visual Personality

- **留白是主要设计元素**——内容密度极低，每个区块周围至少 40-60px 呼吸空间
- **不使用边框和分隔线**——靠留白分隔内容
- **极端字重对比**——700 标题 vs 400 正文，标题不靠 size 靠 weight 突出
- **圆角 12-16px**，柔和但不过分
- **大面积纯色区域**——hero 区域可以全黑/全白，像画册跨页

## Do

- 留白 > 内容，每个元素周围留充足空间
- 标题用加粗但不巨大——靠 weight 而非 size 建层级
- Apple 蓝只用于链接和品牌标识
- 深色区域用纯黑 #000000（Apple 是少数允许纯黑的品牌）
- 文案极少——像广告语，不是段落

## Don't

- 不拥挤——如果内容太多，Apple 风格不是正确选择
- 不用卡片网格——用开放布局
- 不加边框、阴影、渐变
- 不用多色——Apple 的世界只有黑白蓝
- 不在浅色区域用 #000000 做背景——只在深色 hero 区
