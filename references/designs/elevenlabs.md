# ElevenLabs

暗色极简AI语音美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #000000 | 纯黑的AI空间 |
| Surface-1 | #0a0a0a | 微亮的层次层 |
| Surface-2 | #151515 | 次级元素层 |
| Hairline | #222222 | 隐约的分割线 |
| Accent | #1d1d1d | 极致的暗色高光 |
| Ink | #ffffff | 纯净的白色文字 |
| Ink-muted | #b0b0b0 | 柔和的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 600，如声音波形般简洁。正文 DM Sans，字重 400，清晰传达。整体如音频波形图般极简，暗色中见高级。

## Visual Personality

- **极致暗色**——接近纯黑的深邃空间
- **电影质感**——如高端音频软件界面
- **单色克制**——几乎全黑，点缀极少
- **音频美学**——波形与声波的视觉化
- **AI神秘**——技术感的暗色氛围

## Do

- 保持极暗色调，接近纯黑
- 暗色高光仅用于关键交互
- 模仿音频软件的界面布局
- 使用波形或声音相关视觉元素

## Don't

- 使用任何彩色装饰
- 破坏单色调的暗色沉浸感
- 过度使用暗色高光
- 添加非音频相关的视觉元素
