# Together AI

暗色开源AI美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #0a0a0c | 温暖的深色AI空间 |
| Surface-1 | #181820 | 模型与代码层 |
| Surface-2 | #262630 | 次级功能区 |
| Hairline | #343440 | AI分割线 |
| Accent | #3070c0 | 克制的蓝色强调 |
| Ink | #f7f5f0 | 温暖的米白文字 |
| Ink-muted | #a0a090 | 辅助的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 600，如AI平台标题般清晰。正文 DM Sans，字重 400，代码用等宽字体。整体如开源AI平台，技术且专业。

## Visual Personality

- **开源AI**——如开源项目的社区感
- **暗色技术**——开发者友好的暗色
- **蓝色克制**——品牌色极少量点缀
- **AI专业**——人工智能的专业氛围
- **极简理性**——去除所有装饰元素

## Do

- 使用开源平台的界面布局
- 保持温暖的暗色背景
- 蓝色仅用于关键AI术语，极少量
- 模仿开源项目的视觉层次

## Don't

- 过度使用蓝色破坏专业感
- 使用非AI相关的彩色装饰
- 破坏开源平台的极简布局
- 失去AI技术的专业权威感
