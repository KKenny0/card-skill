# Runway

暗色电影工业美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #000000 | 纯黑的电影幕布 |
| Surface-1 | #1a1a1a | 时间轴与编辑层 |
| Surface-2 | #2a2a2a | 次级功能区 |
| Hairline | #333333 | 精准的分割线 |
| Accent | #ffffff | 电影感的白色光 |
| Ink | #ffffff | 清晰的白色文字 |
| Ink-muted | #b0b0b0 | 辅助的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 700，如电影标题般有力。正文 DM Sans，字重 400，如字幕般清晰。整体如视频编辑软件般专业，暗色中见创造力。

## Visual Personality

- **电影工业**——如专业后期软件界面
- **纯色单调**——只有黑与白的极致
- **高对比度**——黑白分明，如电影画面
- **视频美学**——时间轴与帧的视觉化
- **暗色专业**——技术感的暗色工作台

## Do

- 保持极高对比度的黑白
- 白色用于核心交互元素
- 模仿视频编辑软件布局
- 使用时间轴或帧相关视觉

## Don't

- 使用任何彩色装饰
- 降低黑白对比度
- 破坏电影感的暗色调
- 添加非视频相关的视觉元素
