# Framer

暗色创意工作台美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #090909 | 深灰的创意画布 |
| Surface-1 | #1a1a1a | 模块与区块层 |
| Surface-2 | #2a2a2a | 次级功能区 |
| Hairline | #3a3a3a | 几何分割线 |
| Accent | #ffffff | 创意的白色灵感 |
| Ink | #ffffff | 清晰的白色文字 |
| Ink-muted | #b0b0b0 | 辅助的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 700，字间距 -0.01em，营造几何感。正文 DM Sans，字重 400，清晰可读。如设计工具般精准，充满创意张力。

## Visual Personality

- **几何张力**——锐利的线条与形状
- **暗色工作台**——如专业设计软件界面
- **创意自信**——大胆留白与高对比
- **极简克制**——去除一切干扰
- **功能性美学**——每个元素都有目的

## Do

- 使用几何形状和锐利线条
- 白色用于核心交互元素
- 保持暗色为主，白色为辅
- 强调模块化和网格系统

## Don't

- 使用彩色装饰
- 过度使用白色破坏暗调
- 添加复杂的渐变效果
- 失去工具般的精准感
