# Figma

极简设计工具美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #fafaf8 | 温暖的画布纸张 |
| Surface-1 | #f0f0e8 | 图层与工具层 |
| Surface-2 | #e6e6d8 | 次级功能区 |
| Hairline | #dcdcc8 | 精准的分割线 |
| Accent | #1a1a18 | 工具感的黑色强调 |
| Ink | #1c1c1a | 清晰的深灰文字 |
| Ink-muted | #8a8a78 | 辅助的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 700，如设计工具标题般有力。正文 DM Sans，字重 400，如界面文字般清晰。整体如专业设计软件，工具感强烈。

## Visual Personality

- **设计工具**——如专业软件界面
- **极简画布**——干净的创作空间
- **工具理性**——每个元素都有功能
- **结构清晰**——图层与网格系统
- **单色基础**——黑白为主，彩色点缀

## Do

- 使用设计软件的界面布局
- 保持温暖的白色背景
- 黑色用于核心工具或操作
- 使用极少量彩色色块点缀

## Don't

- 过度使用彩色破坏工具感
- 破坏画布的清晰结构
- 添加非工具相关的装饰
- 失去设计软件的专业感
