# Raycast

暗色高效启动器美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #0a0a0a | 温暖的深色启动屏 |
| Surface-1 | #18181a | 搜索与命令层 |
| Surface-2 | #262628 | 次级功能区 |
| Hairline | #343436 | 启动器分割线 |
| Accent | #c45040 | 克制的红色点缀 |
| Ink | #f7f5f0 | 温暖的米白文字 |
| Ink-muted | #a0a090 | 辅助的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 600，如启动器标题般清晰。正文 DM Sans，字重 400，如命令列表般易读。整体如命令启动器，快速且高效。

## Visual Personality

- **启动器暗色**——如命令行启动器
- **高效快速**——生产力工具的流畅感
- **红色克制**——品牌色极少量点缀
- **开发者导向**——为程序员设计
- **极简理性**——去除所有干扰元素

## Do

- 使用启动器的界面布局
- 保持温暖的暗色背景
- 红色仅用于关键命令，极少量
- 模仿启动器的视觉层次

## Don't

- 过度使用红色破坏高效感
- 使用非启动器相关的彩色装饰
- 破坏启动器的极简布局
- 失去生产力工具的流畅感
