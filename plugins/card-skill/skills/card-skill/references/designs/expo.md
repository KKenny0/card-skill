# Expo

极简技术文档美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #fafaf8 | 温暖的文档纸张 |
| Surface-1 | #f0f0e8 | 代码块与示例层 |
| Surface-2 | #e6e6d8 | 导航与次级区 |
| Hairline | #dcdcc8 | 代码分隔线 |
| Accent | #1a1a18 | 技术感的黑色强调 |
| Ink | #1c1c1a | 温暖的深灰文字 |
| Ink-muted | #8a8a78 | 辅助的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 600，如技术文档标题般严谨。正文 DM Sans，字重 400，清晰传达技术信息。代码用等宽字体，如源代码般精准。

## Visual Personality

- **技术文档**——如规范文档般严谨
- **基础设施**——底层框架的稳定感
- **温暖纸张**——如打印文档的质感
- **代码美学**——源代码的视觉语言
- **极简克制**——去除所有干扰元素

## Do

- 使用代码块和技术文档布局
- 保持温暖的白色背景
- 黑色用于关键代码或术语
- 强调技术信息的清晰层级

## Don't

- 使用任何彩色装饰
- 破坏文档的结构化布局
- 过度使用黑色破坏简洁
- 添加非技术相关的视觉元素
