# Clay

温暖有机象牙美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #fffaf0 | 温暖的象牙纸张 |
| Surface-1 | #f0ebe0 | 数据与图表层 |
| Surface-2 | #e0dbce | 次级元素层 |
| Hairline | #d0cbc0 | 有机分割线 |
| Accent | #1a1a18 | 极简的黑色强调 |
| Ink | #1c1c1a | 温暖的深灰文字 |
| Ink-muted | #8a8a78 | 柔和的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 600，如数据报告标题般清晰。正文 DM Sans，字重 400-500，如分析文本般易读。整体如数据丰富的工作台，温暖而有机。

## Visual Personality

- **象牙纸张**——如未漂白的棉纸
- **有机温暖**——自然而非冷感的质感
- **数据丰富**——数据可视化的专业感
- **单色克制**——近单色，温暖为主
- **分析工具**——数据分析的友好氛围

## Do

- 使用数据丰富的界面布局
- 保持温暖的象牙色背景
- 黑色用于关键数据，极少量强调
- 模仿数据工具的视觉层次

## Don't

- 使用任何彩色装饰
- 破坏象牙纸张的温暖质感
- 过度使用黑色破坏简洁
- 添加非数据相关的视觉元素
