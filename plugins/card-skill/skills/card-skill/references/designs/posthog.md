# PostHog

温暖愉悦分析美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #eeefe9 | 温暖的浅绿纸张 |
| Surface-1 | #e0e1d9 | 图表与数据层 |
| Surface-2 | #d2d3c9 | 次级元素层 |
| Hairline | #c4c5b9 | 图表分割线 |
| Accent | #c49020 | 愉悦的琥珀色点缀 |
| Ink | #23251d | 温暖的深绿文字 |
| Ink-muted | #787a6e | 柔和的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 700，如数据仪表盘标题般有力。正文 DM Sans，字重 400-500，如分析报告般清晰。整体如数据工具，温暖而专业。

## Visual Personality

- **温暖浅绿**——如数据分析的舒适空间
- **愉悦活泼**——友好的产品氛围
- **琥珀克制**——品牌色极少量点缀
- **数据可视化**——图表与指标的视觉
- **分析工具**——产品分析的专业感

## Do

- 使用数据仪表盘的界面布局
- 保持温暖的浅绿背景
- 琥珀色仅用于关键数据点，极少量
- 模仿分析工具的视觉层次

## Don't

- 过度使用琥珀色破坏温暖感
- 使用非数据相关的彩色装饰
- 破坏分析工具的结构化布局
- 失去产品分析的专业感
