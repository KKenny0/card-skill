# Cursor

温暖编辑器代码美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #f7f7f4 | 温暖的编辑器背景 |
| Surface-1 | #ebebe4 | 代码块与面板层 |
| Surface-2 | #deded4 | 次级功能区 |
| Hairline | #d1d1c4 | 编辑器分割线 |
| Accent | #c45a20 | 克制的橙色点缀 |
| Ink | #26251e | 温暖的深灰文字 |
| Ink-muted | #7a7a6e | 柔和的灰色文字 |

## Typography Vibe

标题用 DM Sans，字重 600，如编辑器标签页般清晰。正文 DM Sans，字重 400，代码用等宽字体。整体如代码编辑器，温暖而专业。

## Visual Personality

- **编辑器温暖**——如舒适的代码空间
- **AI赋能**——智能助手的友好感
- **橙色克制**——品牌色极少量点缀
- **代码美学**——源代码的视觉语言
- **开发者友好**——为程序员设计

## Do

- 使用代码编辑器的界面布局
- 保持温暖的米白背景
- 橙色仅用于AI提示或关键点
- 模仿编辑器的视觉层次

## Don't

- 过度使用橙色破坏温暖感
- 使用非代码相关的彩色装饰
- 破坏编辑器的结构化布局
- 添加非编程相关的视觉元素
