# OpenCode

终端代码源码美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #faf9f6 | 温暖的终端背景 |
| Surface-1 | #eae8e1 | 代码与面板层 |
| Surface-2 | #dad8cf | 次级元素层 |
| Hairline | #cac8bd | 终端分割线 |
| Accent | #1e1c1a | 终端感的黑色强调 |
| Ink | #1e1c1a | 温暖的深灰文字 |
| Ink-muted | #8a8a78 | 辅助的灰色文字 |

## Typography Vibe

标题用 Berkeley Mono，如终端标题般极客。正文 Berkeley Mono，字重 400，如源代码般精准。整体如IDE界面，单色且专业。

## Visual Personality

- **终端美学**——如命令行界面般极客
- **源码质感**——如阅读源代码般纯粹
- **单色理性**——黑白灰的极简世界
- **开发者导向**——为程序员设计
- **极客克制**——去除所有非必要元素

## Do

- 使用终端或IDE的界面布局
- 全部使用等宽字体
- 黑色用于关键命令或代码
- 模仿终端的视觉层次

## Don't

- 使用任何彩色装饰
- 破坏单色终端的纯粹感
- 使用非等宽字体
- 添加非代码相关的视觉元素
