# Intercom

温暖奶油信件美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #f5f1ec | 温暖的奶油纸张 |
| Surface-1 | #e9e5de | 对话与消息层 |
| Surface-2 | #ded9d0 | 次级元素层 |
| Hairline | #d2cdc2 | 信件般的分割线 |
| Accent | #1a1a18 | 信纸感的黑色文字 |
| Ink | #1e1e1c | 温暖的深灰文字 |
| Ink-muted | #8a8a78 | 柔和的灰色文字 |

## Typography Vibe

标题用 DM Serif Display，如信件标题般优雅。正文 DM Sans，字重 400，如手写信件般自然。整体如优质纸上的手写信，温暖而亲切。

## Visual Personality

- **温暖纸张**——如奶油色优质纸张
- **信件质感**——如手写信件般亲切
- **对话友好**——如聊天窗口般自然
- **人性温度**——温暖而可亲近
- **单色纯粹**——黑与白的优雅对比

## Do

- 使用聊天界面的对话布局
- 保持温暖的奶油色背景
- 黑色用于核心文字，极少量强调
- 模仿信件的视觉温暖感

## Don't

- 使用任何彩色装饰
- 破坏信件的温暖纸张质感
- 过度使用黑色破坏简洁
- 添加非对话相关的视觉元素
