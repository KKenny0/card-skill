# Claude (Anthropic)

暖米色画布，衬线标题，人文感。视觉语言：精装书封面、文学杂志、学术季刊——有智力温度的设计。

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #faf6f0 | 暖米纸张（羊皮纸感） |
| Surface-1 | #f2ebe0 | 提升面板、卡片底色 |
| Surface-2 | #e8dfd2 | 标签背景、次级面板 |
| Hairline | #d4c9b8 | 温暖的分隔线 |
| Accent | #d97757 | 陶土橙 — 品牌色，克制使用 |
| Ink | #2c2418 | 深褐墨主文字 |
| Ink-muted | #7a6f5f | 次要文字、标注 |
| Accent-secondary | #8b7355 | 暗金棕，辅助色 |

## Typography Vibe

衬线标题 + 无衬线正文混排。标题用衬线体（如 Tiempos、Georgia），正文用干净无衬线。排版感觉：书页、报刊——需要阅读的设计。

推荐 web 字体：DM Serif Display（标题）+ DM Sans（正文）。标题 weight 400（衬线体不需要粗体），正文 weight 400。标题 letter-spacing 正常或微正。

## Visual Personality

- **纸张底色是灵魂**——#faf6f0 暖米色像未漂白的纸张
- **衬线标题**——Claude 是少数推荐衬线标题的品牌
- **陶土橙极度克制**——只用于 1 处品牌标记或关键强调
- **内容密度中等**——有呼吸感但不像 Apple 那么空
- **圆角 6px**——温和但不过分
- **留白偏大**——像书籍版心，两侧边距充足

## Do

- 衬线体做标题，无衬线做正文——混排是核心特征
- 陶土橙只用于一处品牌命中
- 用暖灰色调制造层级（不是冷灰）
- 正文字色用深褐墨（#2c2418），不是纯黑
- 背景 sensation 像纸张，不是屏幕

## Don't

- 不用冷色调——整个画面应该在暖灰/米色系内
- 不用纯黑 #000000 做文字——用深褐
- 不用无衬线做标题（除非内容极度技术化）
- 不加阴影——纸面上没有阴影
- 不用高饱和度颜色——像褪色的印刷品
