# Linear

Near-black canvas (#010102, faint blue tint). Lavender accent used with extreme restraint. Depth via surface ladder, never shadows. Feels like software-craft documentation — dense, technical, quietly luxurious.

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #010102 | 画布底色，接近纯黑但带蓝调 |
| Surface-1 | #0f1011 | 卡片、面板底色 |
| Surface-2 | #141516 | 提升的面板、hover 态 |
| Hairline | #23252a | 1px 边框、分隔线 |
| Accent | #5e6ad2 | 薰衣草蓝 — 仅用于 1-2 处强调 |
| Ink | #f7f8f8 | 主文字 |
| Ink-muted | #8a8f98 | 次要文字、标注 |

## Typography Vibe

几何感无衬线体，display 用激进的负字距（tight tracking），body 正常。Weight 梯度：600 标题 / 400 正文。排版感觉：engineered, not editorial。

推荐 web 字体：Inter（weight 500-600 display, 400 body）。大标题可加 letter-spacing: -0.03em。

## Visual Personality

- **深度靠 surface 梯度**，不用阴影。Canvas → Surface-1 → Surface-2 三级跳。
- **1px hairline 边框**定义卡片边界，不用粗线或阴影。
- **单一强调色**——薰衣草蓝只用 1-2 处精确命中，不装饰性使用。
- **圆角 12px** 用于卡片，8px 用于小元素。
- **内容留白大于元素间距**——让信息有呼吸感。

## Do

- 薰衣草蓝只用于：品牌标识、一处关键强调、焦点信号
- 大标题用负字距（-0.03em 到 -0.05em）
- 用 surface 梯度制造层级，而非阴影
- 卡片用 1px hairline 边框

## Don't

- 不加阴影——Linear 用 surface 梯度替代
- 不用薰衣草蓝做背景填充
- 不引入第二强调色
- 不用 #000000 纯黑——canvas 是 #010102（有微蓝调）
- 不把所有卡片做成相同大小——打破对称
