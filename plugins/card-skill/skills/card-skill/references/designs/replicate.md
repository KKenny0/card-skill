# Replicate

温暖技术期刊美学

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #f9f7f3 | 温暖的期刊纸张 |
| Surface-1 | #ebe9e1 | 代码与模型层 |
| Surface-2 | #dddcd1 | 次级元素层 |
| Hairline | #cfcfbd | 期刊分割线 |
| Accent | #b83a20 | 克制的红色点缀 |
| Ink | #222018 | 温暖的深灰文字 |
| Ink-muted | #8a8a78 | 柔和的灰色文字 |

## Typography Vibe

标题用 DM Serif Display，如期刊文章标题般专业。正文 DM Sans，字重 400，代码用等宽字体。整体如技术期刊，温暖而学术。

## Visual Personality

- **期刊质感**——如技术杂志般专业
- **温暖纸张**——如印刷期刊的质感
- **红色克制**——品牌色极少量点缀
- **ML技术**——机器学习的专业氛围
- **学术严谨**——如研究论文般结构化

## Do

- 使用技术期刊的文章布局
- 保持温暖的米白背景
- 红色仅用于关键术语，极少量
- 模仿期刊的视觉层次结构

## Don't

- 过度使用红色破坏温暖感
- 使用非技术相关的彩色装饰
- 破坏期刊的结构化排版
- 添加非ML相关的视觉元素
