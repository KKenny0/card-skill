# Stripe

温白画布（#0a2540 深蓝墨 + #f6f9fc 冰纸底）。品牌色极少使用，靠字重和留白制造层级。视觉语言：年报、数据表、精算单——信息本身即装饰。

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Canvas | #f6f9fc | 冰蓝纸张底色 |
| Surface-1 | #ffffff | 卡片、面板 |
| Surface-2 | #eef2f7 | hover 态、次级面板 |
| Hairline | #e3e8ee | 分隔线、边框 |
| Accent | #635bff | Stripe 紫 — 仅用于 1 处品牌标识 |
| Ink | #0a2540 | 深蓝墨主文字 |
| Ink-muted | #425466 | 次要文字、标注 |
| Data-positive | #30b566 | 数据正向色 |
| Data-negative | #df1b41 | 数据负向色 |

## Typography Vibe

无衬线几何体，中等字重梯度。标题不尖叫——靠色彩和位置建立层级，不靠放大。排版感觉：精密但不过度工程化，有金融文件的庄重感。

推荐 web 字体：DM Sans（weight 500 标题, 400 正文）。标题可适度负字距 -0.02em。

## Visual Personality

- **深度靠 surface 亮度梯度**，不用阴影。Canvas → Surface-1 → Surface-2 三级跳
- **极细 1px 线条**分隔区域，像表格对位线
- **单一品牌紫**——只在一处精确命中（品牌标识或关键 CTA）
- **数据可视化自然融入**——数字和表格是内容的一部分，不是装饰
- **圆角 6px** 用于卡片，小元素 4px

## Do

- 品牌紫只用于一处品牌标记
- 数据区域用表格/指标网格，像财务报告
- 用留白和 hairline 分隔内容，不用色块
- 正文字色用深蓝墨（#0a2540），不是纯黑

## Don't

- 不加阴影——Stripe 靠亮度梯度
- 不用品牌紫做大面积填充
- 不用渐变背景
- 不把所有数字放大成 hero 数字——有选择地突出
- 不引入第二强调色——数据色（绿/红）不算强调色，是功能色
